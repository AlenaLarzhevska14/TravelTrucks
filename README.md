# 🚐 TravelTrucks

TravelTrucks is a modern web application for camper rental. Users can browse
available campers, filter them by different criteria, view detailed information,
read customer reviews, and submit a booking request.

## ✨ Features

- 🏠 Home page with a promotional banner
- 🚐 Campers catalog
- 🔍 Backend filtering by:
  - Location
  - Camper form
  - Engine type
  - Transmission
- 📄 Camper details page
- 🖼️ Image gallery
- ⭐ Customer reviews
- 📝 Booking form
- 🔄 Infinite pagination (Load More)
- ⏳ Loading indicators
- 🎨 Responsive and clean UI based on the provided Figma design

## 🛠️ Tech Stack

- Next.js
- TypeScript
- TanStack Query
- Axios
- React Hook Form
- CSS Modules
- React Icons
- Swiper

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/YOUR_GITHUB_USERNAME/travel-trucks.git
```

Navigate to the project folder:

```bash
cd travel-trucks
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open:

```
https://travel-trucks-delta-azure-54.vercel.app/
```

## 📦 Build

```bash
npm run build
```
